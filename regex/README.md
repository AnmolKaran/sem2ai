# Regex Challenges

Solutions to a set of regular expression pattern-matching problems (problems 60–69).

## What's in here

`regex4.py` contains compiled regex patterns for constraints like:

- Binary strings that avoid substrings `"010"` or `"101"`
- Words with no repeated characters
- Even-length repeated sequences
- Various word/string constraints

Each pattern is a standalone regex string — no external dependencies.

## How to run

```bash
python regex4.py
```

Or import individual patterns into your own script to test them against inputs.
